🚀 BuildProof

BuildProof is a modern developer platform where developers can showcase real projects, solve problems, and get discovered by recruiters — all in one place.

---

🌟 Features

👨‍💻 Developer Side

- 🧩 Create & showcase projects
- 📊 Track stats (likes, views, problems solved)
- 🔥 Activity feed (LinkedIn-style)
- 🧠 Solve coding problems
- 👤 Dynamic profile pages

---

🏢 Recruiter (HR) Side

- 🔍 Browse developers
- ⭐ Shortlist candidates
- 👀 View full profiles
- 📊 Dashboard overview

---

🖥️ Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🔀 React Router DOM
- 📦 JSON (Mock Data)
- 🚀 Vercel (Deployment)

⚙️ How It Works

1. User signs up (Developer / Recruiter)
2. Developer can:
   - Create projects
   - View dashboard
   - Explore feed
3. Recruiter can:
   - Browse candidates
   - View profiles
   - Shortlist developers
